# WeatherDataTest
A simple web page for displaying weather data.

## Install/Run
 - Download [dotnet 8.0](https://dotnet.microsoft.com/en-us/download)
 - Navigate to `WeatherDataTest/WeatherWebApp/`
 - Run `$ dotnet run` to start the server
 - On any browser go to [localhost:5058](http://localhost:5058)
